```plaintext
BEGIN PROGRAM

// Read a line of input from the standard input
READ input_string

// Initialize a starting index for the loop
SET index TO 0

// Initialize an empty string for the output answer
SET output_string TO ''

// Loop until the index reaches the length of the input string
WHILE index is less than the length of input_string DO
    // Check if the current character is a dot
    IF input_string[index] is equal to '.' THEN
        // Append '0' to the output string
        APPEND '0' to output_string
        // Move to the next character
        INCREMENT index by 1
    
    // Check if the next character is also a dot
    ELSE IF input_string[index + 1] is equal to '.' THEN
        // Append '1' to the output string
        APPEND '1' to output_string
        // Skip the next character as it forms a pair
        INCREMENT index by 2
    
    // If neither condition is true, it means we have a dash
    ELSE 
        // Append '2' to the output string
        APPEND '2' to output_string
        // Skip the next character as it forms a pair
        INCREMENT index by 2
    ENDIF
ENDWHILE

// Output the final string
PRINT output_string

END PROGRAM
```

### Summary of Pseudocode Logic:
1. The program reads a string from the input.
2. It initializes an index for traversing the string and an empty string to store the output.
3. It processes the input string character by character:
   - If it encounters a dot, it appends '0' to the output.
   - If it finds a pair of dots, it appends '1'.
   - If it sees a dash (and hence not following another dot), it appends '2'.
4. Finally, it outputs the resulting string.
