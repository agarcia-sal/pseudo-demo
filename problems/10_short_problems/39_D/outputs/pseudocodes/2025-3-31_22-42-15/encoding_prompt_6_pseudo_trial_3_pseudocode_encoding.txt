```
FUNCTION doMain

    // Read two strings representing scores
    LET firstScoreString = input()
    LET secondScoreString = input()
    
    // Split the input strings into lists of scores
    LET firstScores = split firstScoreString by spaces
    LET secondScores = split secondScoreString by spaces
    
    // Initialize a counter for score differences
    LET scoreDifferenceCount = 0 
    
    // Iterate over the first three scores
    FOR index FROM 0 TO 2 DO
        // Convert string scores to integers
        LET firstScore = convert firstScores[index] to integer
        LET secondScore = convert secondScores[index] to integer
        
        // Check if the scores are different
        IF firstScore IS NOT EQUAL TO secondScore THEN
            // Increment the difference counter
            scoreDifferenceCount = scoreDifferenceCount + 1 
        ENDIF
    ENDFOR

    // Determine if the number of differences is less than 3
    IF scoreDifferenceCount IS LESS THAN 3 THEN
        PRINT "YES"
    ELSE 
        PRINT "NO"
    ENDIF
    
END FUNCTION

// Execute the main function
CALL doMain()
``` 

### Summary of Logic:
1. The function reads two space-separated strings of scores.
2. It splits each string into a list of scores.
3. It compares corresponding scores from both lists, counting how many differ.
4. If fewer than three scores differ, it prints "YES"; otherwise, it prints "NO".
