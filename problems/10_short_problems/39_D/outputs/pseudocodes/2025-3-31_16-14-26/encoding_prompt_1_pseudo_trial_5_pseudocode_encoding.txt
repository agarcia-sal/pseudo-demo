```
FUNCTION main()
    // Read two lines of input, representing two sets of values
    firstInput = READ_USER_INPUT()
    secondInput = READ_USER_INPUT()
    
    // Split the input strings into lists of values
    firstValues = SPLIT_STRING(firstInput)
    secondValues = SPLIT_STRING(secondInput)
    
    // Initialize a counter for differences
    differenceCount = 0
    
    // Loop through the first three elements of both lists
    FOR index FROM 0 TO 2 DO
        // Convert the string values to integers for comparison
        valueFromFirst = CONVERT_TO_INTEGER(firstValues[index])
        valueFromSecond = CONVERT_TO_INTEGER(secondValues[index])
        
        // Check if the values are different
        IF valueFromFirst IS NOT EQUAL TO valueFromSecond THEN
            // Increment the difference counter
            differenceCount = differenceCount + 1
        END IF
    END FOR
    
    // Check the number of differences and print the result
    IF differenceCount IS LESS THAN 3 THEN
        PRINT "YES"
    ELSE
        PRINT "NO"
    END IF
END FUNCTION

// Start the program execution
CALL main()
```

- **Input Handling**: The pseudocode indicates that it reads two lines of input.
- **Splitting Strings**: It explains that the input is split into lists of values.
- **Looping**: A loop iterates through the first three values of both lists.
- **Conditionals**: It checks for equality between values and updates a counter accordingly.
- **Output**: Finally, based on the counter value, it determines which message to print.
