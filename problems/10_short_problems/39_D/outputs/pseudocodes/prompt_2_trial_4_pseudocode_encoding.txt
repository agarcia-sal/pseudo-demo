**Pseudocode:**

1. **Start of the Program**
  
2. **Define the main function `doMain`**

3. **Initialize `firstInput` and `secondInput` to get user input**
   - Prompt the user to enter a string of numbers (firstInput)
   - Prompt the user to enter another string of numbers (secondInput)

4. **Split each input string into a list**
   - Create a list `firstList` by splitting `firstInput` into separate numbers
   - Create a list `secondList` by splitting `secondInput` into separate numbers

5. **Initialize a variable `differenceCount` to zero**
   - This variable will count the number of positions where the values differ between the two lists

6. **For each position from 0 to 2 do the following:**
   - Convert the number at the current position in `firstList` to an integer and store it in `firstNumber`
   - Convert the number at the current position in `secondList` to an integer and store it in `secondNumber`
   - If `firstNumber` is not equal to `secondNumber`, increase `differenceCount` by 1

7. **After comparing all positions, check the value of `differenceCount`:**
   - If `differenceCount` is less than 3, output "YES"
   - Otherwise, output "NO"

8. **End of the `doMain` function**

9. **Check if the script is being run directly:**
   - If so, call the `doMain` function to execute the program

10. **End of the program** 

**Input Requirements:**
- Two space-separated strings of three numbers each.

**Output Requirements:**
- Display "YES" if the two inputs differ in less than three positions, otherwise display "NO".
