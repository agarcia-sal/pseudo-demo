```
FUNCTION doMain()
    // Get user input for two sets of three values
    SET firstInput TO input from user
    SET secondInput TO input from user

    // Split the inputs into lists of values
    SET firstValues TO split firstInput into a list of strings
    SET secondValues TO split secondInput into a list of strings

    // Initialize the mismatch counter
    SET mismatchCount TO 0 

    // Compare the corresponding values from both lists
    FOR index FROM 0 TO 2 DO
        // Convert the values to integers for comparison
        SET firstValue TO convert firstValues[index] to integer
        SET secondValue TO convert secondValues[index] to integer

        // Check if the values do not match
        IF firstValue is NOT equal to secondValue THEN
            // Increment the mismatch counter
            INCREMENT mismatchCount by 1 
        END IF
    END FOR

    // Determine if the number of mismatches is less than 3
    IF mismatchCount is less than 3 THEN
        OUTPUT "YES"
    ELSE
        OUTPUT "NO"
    END IF
END FUNCTION

// Main execution starts here
IF this script is run as the main program THEN
    CALL doMain()
END IF
```

### Explanation of Logic:
- The program compares two sets of three values provided by the user.
- It counts the number of mismatched values between the two sets.
- If there are fewer than three mismatches, it outputs "YES"; otherwise, it outputs "NO".
