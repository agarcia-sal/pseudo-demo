``` 
FUNCTION findTriangleNumberIndex()
    // Get the absolute value of the user input as an integer
    SET inputValue TO absolute value of integer(input())
    
    // Initialize the index variable
    SET index TO 0
    
    // Infinite loop to find the triangle number index
    WHILE true DO
        // Calculate the ith triangle number using the formula
        SET triangleNumber TO (index * (index + 1)) / 2
        
        // Calculate the difference between the triangle number and the input value
        SET difference TO triangleNumber - inputValue
        
        // Check if the triangle number equals the input value
        IF triangleNumber EQUALS inputValue THEN
            // Print the index if a match is found
            PRINT index
            BREAK // Exit the loop
            
        // If the triangle number exceeds the input value
        ELSE IF triangleNumber GREATER THAN inputValue THEN
            // Check if the difference is even
            IF difference MODULO 2 EQUALS 0 THEN
                // Print the index if the difference is even
                PRINT index
                BREAK // Exit the loop
        END IF
        
        // Increment the index for the next iteration
        index INCREMENT BY 1
    END WHILE
END FUNCTION
```

### Explanation:
1. **Input Handling:** The function reads an integer input and converts it into its absolute value to ensure the calculation is valid for positive numbers.
  
2. **Index Calculation:** A while-loop continuously checks for the index of the triangle number that matches or exceeds the input value. 

3. **Triangle Number Logic:** Triangle numbers are computed using the formula (n * (n + 1)) / 2. The function checks if this number equals the user input or if it is greater and if the difference is even.

4. **Output:** The index is printed when one of the conditions is met, and the loop is exited.
