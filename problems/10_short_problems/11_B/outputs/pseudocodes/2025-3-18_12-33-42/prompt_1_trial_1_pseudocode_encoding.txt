BEGIN

    // Read a value from user input, convert it to an integer, and take its absolute value
    userInput = ABSOLUTE_VALUE(READ_INTEGER())
    index = 0

    // Infinite loop that continues until a break condition is met
    WHILE TRUE DO

        // Calculate the sum of the first 'index' natural numbers
        sumOfNaturalNumbers = (index * (index + 1)) / 2
        
        // Determine how much greater the sum is than the user input
        difference = sumOfNaturalNumbers - userInput
        
        // Check if the sum equals the user input
        IF sumOfNaturalNumbers EQUALS userInput THEN
            PRINT index // Output the current index
            BREAK // Exit the loop
            
        // Check if the sum is greater than the user input
        ELSE IF sumOfNaturalNumbers GREATER THAN userInput THEN
            
            // Check if the difference is even
            IF difference MODULO 2 EQUALS 0 THEN
                PRINT index // Output the current index
                BREAK // Exit the loop
        
        // Increment the index for the next iteration
        index = index + 1

END

### Explanation of Pseudocode Logic:
1. The program starts by taking a user's input, ensuring it is a non-negative integer.
2. It initializes an index variable to zero, which will be used to calculate the sum of natural numbers.
3. The loop runs indefinitely, calculating the sum of natural numbers up to the current index.
4. It checks if this sum equals the user's input. If so, it prints the index and exits.
5. If the sum exceeds the user's input, it checks if the difference is even. If it is, the index is printed and the loop breaks.
6. The loop increments the index and repeats until one of the conditions is satisfied.
