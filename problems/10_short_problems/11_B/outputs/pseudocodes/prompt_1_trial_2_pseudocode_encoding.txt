**Pseudocode: Calculate the Minimum Non-Negative Integer for Specific Sum Conditions**

1. **Input Step:**
   - Receive an integer input from the user.
   - Take the absolute value of this integer and store it in a variable named `targetSum`.

2. **Initialization Step:**
   - Initialize a variable `index` to 0, which will be used to calculate triangular numbers.

3. **Looping Step:**
   - Start an infinite loop:
     - Calculate the triangular number `sumUpToIndex` as the sum of the first `index` integers using the formula: 
       - `sumUpToIndex = (index * (index + 1)) / 2`
     - Calculate the difference `excessAmount`:
       - `excessAmount = sumUpToIndex - targetSum`

     - **Condition Check:**
       - If `sumUpToIndex` is equal to `targetSum`:
         - Print the current value of `index`.
         - Exit the loop.

       - If `sumUpToIndex` is greater than `targetSum`:
         - Check if `excessAmount` is an even number:
           - If `excessAmount` is even:
             - Print the current value of `index`.
             - Exit the loop.

     - **Increment Step:**
       - Increase the value of `index` by 1 and repeat the loop.

4. **Output Step:**
   - The output will be the smallest non-negative integer `index` that meets the specified conditions.
