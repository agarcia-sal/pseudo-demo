**Pseudocode:**

1. **Start**
2. **Get User Input**
   - Convert the user-provided input to an integer.
   - Take the absolute value of this integer and assign it to a variable named `desiredNumber`.

3. **Initialize Variables**
   - Set a counter `index` to 0.

4. **Begin Loop (indefinitely)**
   - Calculate the sum of all integers from 1 to `index` and store it in a variable `currentSum`.
     - The formula to compute this sum is: `currentSum = (index * (index + 1)) / 2`
   
   - Calculate the difference between `currentSum` and `desiredNumber` and store it in `difference`.
     - `difference = currentSum - desiredNumber`
   
   - **Check Conditions**
     - **If `currentSum` equals `desiredNumber`:**
       - Print the value of `index`.  
       - Exit the loop.
     - **Else if `currentSum` is greater than `desiredNumber`:**
       - **If `difference` is even:**
         - Print the value of `index`. 
         - Exit the loop.

   - Increment the `index` by 1.

5. **End Loop**
  
6. **End Program**
