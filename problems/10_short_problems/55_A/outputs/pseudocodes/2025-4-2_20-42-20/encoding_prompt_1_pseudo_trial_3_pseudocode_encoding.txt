**Pseudocode:**

1. **Input**: Read an integer value `n` which represents the number of elements.
2. **Initialize**: 
   - Create a list `isTrue` of size `n` filled with `True` values. This list will be used to track the status of each position.
   - Set an integer variable `currentIndex` to 0.
   - Set an integer variable `step` to 1.
3. **Loop until step exceeds 500,000**:
   - Check if the value at position `currentIndex` in `isTrue` is `True`:
     - If it is `True`, set that position in `isTrue` to `False`.
   - Increment `step` by 1.
   - Update `currentIndex` to the new value using the formula: `currentIndex = (currentIndex + step) mod n`.
4. **Filter**: Create a new list `remainingTrue` that contains all the `True` values from `isTrue`.
5. **Check outcome**:
   - If the length of `remainingTrue` is 0, output "YES".
   - Otherwise, output "NO".
