**Pseudocode:**

1. **Start**
2. **Input:** Read an integer number `n` from user input.
3. **Initialize:** Create a list called `status` with `n` elements, all set to `True`.
4. **Set Counter:** Initialize two variables, `currentStep` set to 1 and `index` set to 0.
5. **Loop:** While `currentStep` is less than or equal to 500000 do the following:
   - **Check Status:** If the element at position `index` in the `status` list is `True`:
     - Set the element at `index` in `status` to `False`.
   - **Increment Steps:** Increase `currentStep` by 1.
   - **Update Index:** Calculate the next index as `(index + currentStep) MOD n`.
6. **Filter Active Status:** Create a new list called `activeElements` that contains all elements from `status` that are still `True`.
7. **Check Result:** 
   - If the length of `activeElements` is 0:
     - Print "YES"
   - Otherwise:
     - Print "NO"
8. **End**
