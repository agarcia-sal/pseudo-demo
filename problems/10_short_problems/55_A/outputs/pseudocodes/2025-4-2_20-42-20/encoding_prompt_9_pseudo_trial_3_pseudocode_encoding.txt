**Purpose:**  
Determine if a specific condition holds true for a generated array based on a given integer input.

---

**Pseudocode:**

1. **Start Program**
   
2. **Accept Input:**
   - Set `size` to the integer value received from user input.

3. **Initialize Array:**
   - Create a boolean array `status` of length `size` and fill it with `True`.

4. **Set Initial Variables:**
   - Set `index` to 0.
   - Set `increment` to 1.

5. **Loop until a condition is met:**
   - While `increment` is less than or equal to 500,000 do the following:
     1. **Check Current Status:**
        - If `status[index]` is True, then:
          - Set `status[index]` to False.
     2. **Update Values:**
        - Increment `increment` by 1.
        - Update `index` to be `(index + increment) modulo size`.

6. **Filter Active Status:**
   - Create a new list `active_status` consisting of all elements from `status` that are still `True`.

7. **Determine If Any True Elements Exist:**
   - If the length of `active_status` is 0, then:
     - Print "YES".
   - Otherwise:
     - Print "NO".

8. **End Program**
