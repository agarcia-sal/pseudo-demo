### Pseudocode

1. **Initialize Input**
   - Read an integer `n`, which represents the size of the list.

2. **Create a Boolean List**
   - Initialize a list `isActive` of size `n`, where all elements are set to `True`.
   
3. **Set Up Variables**
   - Set a variable `index` to `0`.
   - Set a variable `counter` to `1`.

4. **Iterate While Counter is Less Than or Equal To 500000**
   - While `counter` is less than or equal to 500000 do the following:
     - If `isActive` at position `index` is `True`, set that position to `False`.
     - Increment the `counter` by 1.
     - Update `index` to be the result of `(index + counter) modulo n` to wrap around if necessary.

5. **Filter Active Elements**
   - Create a new list `activeElements` containing only the elements in `isActive` that are still `True`.

6. **Check Active Elements**
   - If the length of `activeElements` is `0`, output "YES".
   - Otherwise, output "NO".
