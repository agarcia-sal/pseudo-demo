**Pseudocode:**

1. **Start of the Program**

2. **Input Handling**: 
   - Read an integer value from the user and store it in the variable `numberOfElements`.

3. **Initialization**:
   - Create a list called `statusList` of size equal to `numberOfElements`, initializing all its values to `True`.
   - Set an index variable `currentIndex` to 0.
   - Set a counter variable `stepIncrement` to 1.

4. **Looping Process**: 
   - Repeat the following steps as long as `stepIncrement` is less than or equal to 500,000:
     a. Check the value at `currentIndex` in the `statusList`.
     b. If the value is `True`, set that value to `False`.
     c. Increment `stepIncrement` by 1.
     d. Update `currentIndex` by adding `stepIncrement` to it, then take the result and calculate the remainder when divided by `numberOfElements` (this ensures that `currentIndex` wraps around if it goes past the end of the list).

5. **Filtering Results**:
   - Create a new list called `trueValues` that contains all elements from `statusList` that are still `True`.

6. **Output Handling**:
   - Check the length of `trueValues`:
     a. If the length is 0, print "YES".
     b. Else, print "NO".

7. **End of the Program**
