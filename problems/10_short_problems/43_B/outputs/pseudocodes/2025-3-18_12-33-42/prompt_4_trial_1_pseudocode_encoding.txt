```
BEGIN

  // Read input strings while removing spaces
  string1 = REMOVE_SPACES(GET_USER_INPUT())
  string2 = REMOVE_SPACES(GET_USER_INPUT())

  // Initialize frequency differences array
  frequencyDifferences = INITIALIZE_ARRAY_WITH_SIZE(LETTER_COUNT)

  // Calculate frequency differences for each character from 'A' to 'z'
  FOR each character ASCII value from 'A' (65) to 'z' (122) DO
    frequencyDifferences[character] = COUNT(character IN string1) - COUNT(character IN string2)
  END FOR

  // Check if all frequency differences are non-negative
  IF ALL_VALUES_IN_ARRAY(frequencyDifferences ARE >= 0) THEN
    PRINT "YES"
  ELSE
    PRINT "NO"
  END IF

END
```

### Comments:
- `REMOVE_SPACES()` is a function that cleans the input by eliminating spaces.
- `GET_USER_INPUT()` captures user input.
- `INITIALIZE_ARRAY_WITH_SIZE(LETTER_COUNT)` creates an array (size = 58 for the range A-Z and a-z).
- `COUNT(character IN string)` counts occurrences of `character` in the given `string`.
- `ALL_VALUES_IN_ARRAY(array ARE >= 0)` checks if every element in the array meets the condition.
