**Purpose Summary**: The goal of this code is to convert a string that includes formatted email-like phrases by replacing certain words with symbols and ensuring proper formatting for a valid email structure.

### Pseudocode

1. **Start Program**
   
2. **Read Input**  
   - Get a line of input from the user and remove any extra spaces.

3. **Replace Specific Words**  
   - Replace every occurrence of the word "dot" with the character "."  
   - Replace every occurrence of the word "at" with the character "@"  

4. **Ensure Proper Start Character**  
   - If the first character of the processed string is ".", prepend "dot" to the string.

5. **Initialize Variables**  
   - Set a counter variable `counter` to 0.
   - Create an empty list `characterList` to store modified characters.
   - Initialize a variable `length` to 0 (not used but defined).

6. **Check for Starting "at" Character**  
   - If the first character of the string is "@", replace it with "at" followed by the rest of the string.

7. **Process Each Character in the String**  
   - For each character in the modified string:
     - If the character is:
       - "@"
         - If `counter` is greater than 0:
           - Append "at" to `characterList`.
           - Set `counter` to 1 (indicating "at" has been processed).
         - Else:
           - Append "@" to `characterList`.
           - Set `counter` to 1.
       - Else, append the character to `characterList`.

8. **Combine Characters into a Single String**  
   - Join all elements of `characterList` into a single string called `finalString`.

9. **Ensure Proper End Character**  
   - If the last character of `finalString` is ".", replace it with "dot".

10. **Output the Final Result**  
    - Print the `finalString`.

**End Program**
