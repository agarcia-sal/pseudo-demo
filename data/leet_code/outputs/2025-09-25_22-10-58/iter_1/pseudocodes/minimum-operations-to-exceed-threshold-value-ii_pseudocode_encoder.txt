CLASS Solution  
    FUNCTION minOperations(nums parameter k parameter)  
        CALL heapify ON nums  
        SET operations TO zero  
        WHILE element at position zero of nums LESS THAN k AND the LENGTH OF nums GREATER THAN one  
            SET x TO result of calling heappop ON nums  
            SET y TO result of calling heappop ON nums  
            SET new_value TO the minimum of x and y MULTIPLIED BY two PLUS the maximum of x and y  
            CALL heappush ON nums WITH new_value  
            INCREMENT operations BY one  
        END WHILE  
        RETURN operations  
    END FUNCTION  
END CLASS
